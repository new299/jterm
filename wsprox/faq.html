Minaterm FAQ


General questions

What is this?

Minaterm is a free web-based ssh client. It lets you connect to ssh servers over the Internet via your web browser. If you don't know what ssh is, you probably wont be very interested in Minaterm anyway. But you can read about ssh on wikipedia.

End-to-end encryption you say?

Minaterm uses libssh2 (compiled to Javascript using Emscripten) to create a full ssh client in your browser. Your session data is encrypted in the browser and never touches the Minaterm server in unencrypted form. Unless the client-side Javascript is tampered with by a malicious 3rd party it should not be possible to intercept the traffic on our servers or in flight. However, Minaterm is presently a proof of concept, and I'd advise against using it for anything security sensitive. The hostname to which you want to connect is not encrypted, however Minaterm also supports SSL, and if you connect using HTTPS this information will also be encrypted in flight. The hostname could potentially be intercepted on the Minaterm server, however this information is not currently logged. Outbound connections are currently routed over Tor (except for connections to the local Minaterm server used for the Eliza demo).

Issues

It's very slow

The Javascript itself is reasonably fast, a local install is a little slower than a native terminal, but quite usable. However, the instance running at www.minaterm.com routes all outbound connections over Tor, this will likely cause performance issues (particularly under load). The Eliza demo runs on the local host and should give a better representation of expected performance. This server is also located in Sweden, trans-atlanic/pacific connections may add additional latency. If you'd like me to create a private installation for you, please contact me.

Rendering Issues

There are a few known rendering issues, certain versions of Firefox appear not to render the cursor on an otherwise blank line. A white highlight appears on the left side of the screen on ocassion. This can be fixed by resizing the window. If you notice any other rendering issues please contact me or create a github issue.
