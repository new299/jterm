Minaterm FAQ


General questions

What is this?

Minaterm is a free web-based ssh client. It lets you connect to ssh servers over the Internet via your web browser. If you don't know what ssh is, you probably wont be very interested in Minaterm anyway. But you can read about ssh on wikipedia.

End-to-end encryption you say?

Minaterm uses libssh2 (compiled to Javascript using Emscripten) to create a full ssh client in your browser. Your session data is encrypted in the browser and never touches the Minaterm server in unencrypted form. Unless the client-side Javascript is tampered with by a malicious 3rd party it should not be possible to intercept the traffic on our servers or as in flight. The hostname to which you want to connect is not encrypted, however Minaterm also supports SSL, and if you connect using HTTPS this information will also be encrypted in flight. The hostname could potentially be intercepted on the Minaterm server, however this information is not currently logged. Outbound connections are currently routed over Tor (except for connections to the local Minaterm server used for the Eliza demo).
